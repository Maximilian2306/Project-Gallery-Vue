## Features

- **Dark/Light Mode** - Toggle with system preference detection
- **Multi-Language** - German and English
- **Advanced Filtering** - Filter by languages, frameworks, technologies
- **Real-time Search** - Search by project name or tech stack
- **Responsive Design** - Optimized for all devices
- **Persistent Settings** - Preferences saved to localStorage
- **Categorized Display** - Games, Tools, Services

## Tech Stack

- **Vue 3** - Composition API
- **Vite** - Build tool
- **FontAwesome** - Icons
- **CSS Custom Properties** - Theming
